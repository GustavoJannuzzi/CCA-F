# CCA-F — Apostila Interativa de Estudo

Material de estudo interativo para a certificação **Claude Certified Architect – Foundations (CCA-F)**.

## Como usar

Abra **`index.html`** no navegador. Não precisa de servidor nem build — é uma página única e autocontida.

## O que tem

- **Dashboard** com o mapa mental "onde cada coisa vive" (Claude Code / Agent SDK / Claude API / MCP).
- **5 domínios** cobrindo as 30 task statements oficiais, cada subtópico explicado de forma funcional: onde vive → em uma frase → o cenário → como funciona → quando usar → pegadinha do exame.
- **Cenários do Exame** — os 6 cenários oficiais.
- **Simulado Interativo** — questões no estilo da prova, com explicação da resposta certa e das erradas.
- **Referência Rápida** — cheat sheet de distinções críticas.
- **Exam Topics (oficial)** — o blueprint do exame em português.

## Estrutura do exame

60 questões · 120 minutos · aprovação **720/1000** · 4 dos 6 cenários sorteados por prova.

| Domínio | Peso |
|---|---|
| 1 · Agentic Architecture & Orchestration | 27% |
| 3 · Claude Code Configuration & Workflows | 20% |
| 4 · Prompt Engineering & Structured Output | 20% |
| 2 · Tool Design & MCP Integration | 18% |
| 5 · Context Management & Reliability | 15% |
