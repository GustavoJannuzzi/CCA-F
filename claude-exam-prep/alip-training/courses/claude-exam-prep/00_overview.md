## Exam Overview

- **Format**: Multiple choice (4 options, 1 correct)
- **Passing Score**: 720 / 1000
- **No penalty** for guessing
- **4 of 6 scenarios** presented at random per exam sitting

## Exam Scenarios

1. **Customer Support Resolution Agent** - Agent SDK, MCP tools, escalation
2. **Code Generation with Claude Code** - Slash commands, CLAUDE.md, plan mode
3. **Multi-Agent Research System** - Coordinator-subagent, context passing
4. **Developer Productivity with Claude** - Built-in tools, MCP servers
5. **Claude Code for CI/CD** - Pipelines, structured output, automated reviews
6. **Structured Data Extraction** - JSON schemas, validation, batch processing
