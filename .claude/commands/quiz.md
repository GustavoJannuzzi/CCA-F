Inicie uma sessão de quiz interativa para praticar para o exame CCA-F.

Modo de operação:
1. Pergunte quantas questões quero praticar (sugerir 10)
2. Pergunte se quero focar em um domínio específico ou todos
3. Para cada questão:
   - Indique o número e domínio (ex: "Questão 3/10 — Domínio 2: Tool Design")
   - Apresente um cenário realista de produção
   - Faça a pergunta
   - Liste 4 opções (A, B, C, D)
   - Aguarde minha resposta
   - Mostre se acertei ✅ ou errei ❌
   - Explique POR QUE a resposta correta está certa
   - Explique por que CADA opção incorreta está errada
4. No final, mostre: score total, breakdown por domínio, e 3 conceitos para revisar

Foco especialmente em questões que testam julgamento prático sobre trade-offs (programmatic enforcement vs. prompt-based, hooks vs. prompt instructions, batch vs. synchronous API, etc.).

Baseie as questões nos 6 cenários do exame: Customer Support Agent, Code Generation with Claude Code, Multi-Agent Research System, Developer Productivity, Claude Code for CI/CD, e Structured Data Extraction.
